准备工作#

这里需要使用到一个叫做github page的东西，是GitHub提供的一个网页寄存服务，可以寄存静态页面，这个特性就特别适合博客的搭建。

首先再Github上创建一个 Repository，命名为Github名字xxx.github.io，譬如我的仓库lijiejoy.github.io，这样就可以生成一个用户页面.

在hugo server调式页面完成后，使用命令hugo生成hugo静态页面。

注意！！！在生成静态页面之前要把config.toml文件中的baseURL修改为自己博客的网址，譬如

1
2

baseURL = "https://lijiejoy.github.io/"
hugo    ##生成静态页面文件

在命令执行后，生成一个public文件夹，里面就是网站的静态页面文件

首次上传#

进入public文件夹，使用git上传文件

1
2
3
4
5
6

cd public   ##进入静态资源的目录
git init    ##初始化仓库
git remote add origin https://github.com/lijiejoy/lijiejoy.github.io.git    ##链接远程仓库
git add .
git commit -m "first commit" 
git push -u origin master ##上传静态文件资源
